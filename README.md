# 🎓 Student Management System (Version 2)

A simple **Student Management System** built in **C** using **Arrays, Functions, Loops, and Strings**. This project performs basic CRUD (Create, Read, Update, Delete) operations on student records through a menu-driven interface.

---

## 🚀 Features

- ➕ Add Student
- 👀 Show All Students
- 🔍 Search Student by Roll Number
- ✏️ Update Student Details
- 🗑️ Delete Student
- 🚫 Duplicate Roll Number Validation
- 🧑 Store Student Name, Age, Marks, and Roll Number
- 📋 Menu Driven Program

---
